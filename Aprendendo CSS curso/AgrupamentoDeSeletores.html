Agrupamento de seletores

Podemos aplicar as mesmas regras CSS para varios seletores diferents.Para isso, podemos separar os seletores utilizando uma virgula(,).

Assim, não precisamos ficar copiando e colando os mesmo codigos para todos, nós podemos fazer um só bloco de css e reutilizar a mesma regra!
