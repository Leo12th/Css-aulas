Seletores CSS

Bem-Vindo!

No CSS nós possuimos os seletores por id

  • Seletor por tipo:Busca elementos por uma tag HTML
  • Seletor por id(#):Busca elementos atraves do atributo "id"
  • Seletor por Class(.):Busca elementos atraves do atributo "class"
  • Seletor Universal:Seleciona todos os elementos do HTML
  • Seletor de atributo ([atrib]):Seleciona elementos que possuem um atributo especifico em nosso documento html e conseguimos buscar atributos com um valor especifico
  • Seletor por tipo:Busca elementos por uma tag HTML
